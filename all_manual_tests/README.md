# All Manual Test Cases

This folder contains all manual test cases collected from various projects.  
Each test case follows a structured format and includes:

- Test Case ID, Title, and Module  
- Preconditions  
- Test Steps  
- Expected Result  
- Actual Result  
- Notes (if applicable)
